失眠和抑郁有关系吗？

无忧睡眠 · 发布于 2026-06-16

标题：失眠3个月没好，你排查过这件事吗？

> 很多人把失眠当成一个独立的问题来治——

>

> 试遍褪黑素、泡脚、白噪音、运动……

>

> 用了很多方法，就是好不了。

>

> 如果你也是这种情况，有一件事值得认真面对：

>

> 你的失眠，有没有可能不只是失眠？

失眠和抑郁的关系，没有你想的那么简单

很多人以为：先有失眠，睡不好了，情绪才变差。

事实上，两者互为因果，形成闭环：

```

睡不好 → 情绪变差、精力下降 → 更焦虑失眠 → 睡得更差 → 情绪更差...

```

研究表明：

  • 慢性失眠的人，发展成抑郁的概率是正常人的2-3倍
  • 抑郁症患者中，有80-90%存在睡眠问题（入睡困难、早醒、睡眠浅）

两者太常共病，以至于临床上有一个专有名词叫"共病失眠"。

怎么判断你的失眠是否已经伴随抑郁倾向？

以下8个问题，对照一下自己：

| # | 问题 | 你的情况 |

|---|---|---|

| 1 | 早上4-5点自动醒，醒来心情很差，想到今天就觉得没意思 | □ 有 □ 没有 |

| 2 | 对曾经喜欢的事情没兴趣了 | □ 有 □ 没有 |

| 3 | 整天有说不清楚的疲惫感，和睡眠时间无关 | □ 有 □ 没有 |

| 4 | 注意力和记忆力明显下降，工作经常出错 | □ 有 □ 没有 |

| 5 | 情绪持续低落超过两周，不是因为某件具体的事 | □ 有 □ 没有 |

| 6 | 有"活着没意思""不想和人说话"的念头 | □ 有 □ 没有 |

| 7 | 食欲明显变差或暴增 | □ 有 □ 没有 |

| 8 | 对未来感到很绝望，不知道为什么要努力 | □ 有 □ 没有 |

如果你有3个及以上"有"，建议认真对待——你的失眠可能需要从情绪角度介入。

如果有第6条，建议尽快寻求专业支持。

失眠+抑郁同时存在，怎么办？

首先：不要只治失眠

如果你的失眠是由情绪问题导致的，单治失眠是治标不治本。

就好像一个人发烧，给他退烧药，只是把温度压下去，但感染还在。

失眠+情绪问题，需要同时处理：

  • 睡眠层面：建立规律作息、睡眠卫生习惯
  • 情绪层面：认知调整、情绪疏解、必要时药物辅助

其次：识别"抑郁型早醒"

和"焦虑型睡不着"不同，抑郁相关的失眠有一个特征：早醒。

  • 不是入睡困难，而是凌晨3-5点突然醒，怎么都睡不着
  • 醒来情绪很低落，有时候会哭，不知道为什么
  • 越躺着越压抑

如果你符合这个描述，这是抑郁症的典型睡眠表现，需要认真关注。

第三：可以做的事情

自助层面：

  1. 每天记录情绪和睡眠（简单记录：今天情绪1-10分，睡了几小时，几点醒）
  2. 白天增加户外活动，阳光对情绪改善有明显帮助
  3. 保持社交，不要一个人隔离
  4. 避免饮酒（酒精短期助眠，但会加重抑郁）

专业支持层面：

  • 可以去心理科做一个正式的评估（PHQ-9量表可以初步自测）
  • 认知行为疗法（CBT）对失眠+抑郁共病有明确效果
  • 如果需要药物，配合医生，不要自己乱用或突然停药

我自己的经历

我失眠的最严重阶段，也有过情绪很低落的时期。

当时的状态是：睡不好，白天没精神，对什么都提不起兴趣，觉得自己很没用。

直到做了系统的心理评估，才发现不只是"睡不好"，是焦虑和轻度抑郁倾向在同时影响我。

处理了情绪问题之后，睡眠慢慢就好了。

很多时候，失眠只是冰山一角。

最后说一句

失眠和抑郁不是羞耻的事，是身体在说"我需要帮助了"。

如果你的失眠已经超过3个月，同时情绪也很差，请认真对待——不要硬撑。
